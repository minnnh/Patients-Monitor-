Patient Monitor Platform

Welcome to Patient Monitor !

This website is a Patient Monitor platform, which could realize the functions below.

  • Storage: a function has the attributes of users' 'Premission', 'User_id', 'Device_id', 'Roles'.

        Route of the function:  '/create'

  • Users: a function has the attributes of  users' 'User_id', 'Name', 'Date_of_Birth', 'Roles', 'Gender'. 

        Route of the function:  '/users'

  • Devices: a function has the attributes of users'  'Device_id', 'MAC', 'Date_of_Purchase', 'User_id', 'Fir_ver'. 

        Route of the function:  '/devices'

  • Measurements: a function has the attributes of users' 'User_id', 'Weight', 'Height', 'Temperature', 'Systolic_Pressure', 'Diastolic_Pressure', 'Pulse', 'Oximeter', 'Glucometer'.

        Route of the function:  '/measurements'

  • Assignments: a function has the attributes of users' 'Premission', 'User_id', 'Device_id', 'Roles'.

        Route of the function:  '/assignments'

All of the functions above are able to 1) get the data of the values of its attributes;  2) create new value of its attributes 

  • Records: a function has the attributes of the chat infomation of users. Attributes: 'User_id', 'Connect_id', 'Message_Type', 'Content', 'Time'. And the number of record and the times of the record with the same user will be saved automatically.

        Route of the function:  '/chat'

The functions above is able to 1) get the data of the values of its attributes;  2) create the new values of the chat's attributes in the function.
